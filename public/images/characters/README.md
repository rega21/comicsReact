# Imagen de Ejemplo - Batman

Para probar el sistema de imágenes locales, puedes:

1. Descargar cualquier imagen de Batman
2. Renombrarla como `batman.jpg` 
3. Colocarla en: `public/images/characters/batman.jpg`

Cuando abras la aplicación, verás que Batman usa tu imagen local en lugar del placeholder.

## Ejemplo de búsqueda:
- Google: "Batman character art high resolution"
- Tamaño recomendado: 300x450 píxeles
- Formato: JPG

¡Una vez que veas que funciona con Batman, puedes descargar el resto!
